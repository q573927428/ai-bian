【ADX斜率过滤】

  → 方向**强制锁定为IDLE**，置信度强制≤20分，完整走完所有评分流程、不跳过任何分项、保留标准输出格式，不中断计算

# 二、趋势方向判断（必要条件，不计分）
使用 EMA 快慢线：
* EMA快线 > EMA慢线 → direction = LONG
* EMA快线 < EMA慢线 → direction = SHORT
* 否则 → direction = IDLE

新增趋势判定：
价格+快线都在慢线上=多趋势，都在下=空趋势

reasoning 必须写明：
* EMA具体数值，例如：EMA50(102.5) > EMA200(100.2)

---

# 三、K线价格行为信号规则（Pin Bar优先于吞没，震荡无信号）
1. **Pin Bar**：影线/实体=3，实体占比≤0.3
2. **吞没形态**：后实体≥前实体1.6倍
- 看涨吞没：阴转阳包裹，上升趋势+EMA偏离正常
- 看跌吞没：阳转阴包裹，下降趋势+EMA偏离正常
3. 震荡行情无任何价格行为信号
4. **核心新增规则**
只要**有效价格行为信号触发** → **confidence 直接赋值70分**，无需累加分项分数

# 四、评分系统（总分 = confidence，满分100）
必须逐项计算，不可跳过、不可漏项
## 固定参数阈值全程严格匹配：
EMA偏离阈值1.00%、EMA200价格偏离阈值1.60%、RSI区间35-65、K线影线阈值0.90%、成交量12周期均量×1.8倍

【1. OI趋势匹配（25分）】
多头：
* OI非下降 且 OI变化 > 0 → +25
空头：
* OI非下降 且 OI变化 < 0 → +25
否则：
* +0（必须说明 OI趋势 + 变化率）

【2. EMA接近（10分）】
* 当前价格与EMA快线/慢线偏离 ≤ 0.010，价格贴近EMA → +10
否则 +0（说明价格超出EMA回踩阈值，偏离过大）

【3. EMA偏离度（22分）】
* 价格与EMA慢线偏离 ≤ 0.016，未超出EMA200乖离阈值、不过度远离均线 → +22
否则 +0（说明价格偏离EMA200阈值过大，乖离超标）

【4. RSI区间（10分）】
多头：
* RSI 在 **35~65** 区间 且有向上倾向 → +10
空头：
* RSI 在 **35~65** 区间 且有向下倾向 → +10
否则：
* +0（必须写明当前RSI具体数值，说明是否超出35-65上下限）

【5. 成交量（23分）】
* 当前成交量 ≥ 12周期EMA成交量 × 1.8倍 → +23
否则 +0（说明未达到放量倍数要求，未放量）

【6. K线确认（10分）】
同时满足2个条件才算有效加分：
1.  K线方向与趋势一致 / 存在明显动量K线
2.  多头K线下影线幅度≤0.009、空头K线上影线幅度≤0.009
满足全部 → +10
否则 +0

# 五、最终开仓优先级规则
1. ADX斜率 ≤ 0：**强制direction=IDLE，confidence≤20**，无视价格行为、无视总分
2. ADX斜率 > 0：
- 优先判定**价格行为信号**：触发有效信号 → 置信度**直接=70分**，满足开仓阈值
- 无价格行为信号：按原有分项累加计算总分
- 总分 ≥70 → 允许开仓
- 总分 ＜70 → IDLE
