激进短线交易策略（带明确direction=LONG/SHORT）
核心目标：捕捉市场启动和动量机会，不等待完美确认，优先把握即时动量

## 一、市场状态判定
1. **多头趋势**：EMA7＞EMA20＞EMA50 顺向多头排列
2. **空头趋势**：EMA7＜EMA20＜EMA50 顺向空头排列
3. **震荡行情**：EMA7/20/50相互缠绕、无明显有序排列
4. **趋势启动**：价格刚向上突破EMA20/EMA50 → 多头启动；价格刚向下跌破EMA20/EMA50 → 空头启动

## 二、入场触发信号（满足任意1项即可）
1. 价格向上突破EMA20/EMA50
2. 价格向下跌破EMA20/EMA50
3. 出现大实体动量单边K线
4. 价格回踩EMA20后强势反弹
5. 价格反弹触碰EMA20后承压回落

## 三、多空分项评分（每项+1分）
### 多头 LONG 评分项
- RSI处于30~50区间且持续向上
- 持仓量OI同步增加
- 成交量明显放大
- EMA多头排列/多头启动结构

### 空头 SHORT 评分项
- RSI处于50~70区间且持续向下
- 持仓量OI同步增加
- 成交量明显放大
- EMA空头排列/空头启动结构

## 四、开仓方向与执行规则
1. 多头触发成立 + 总分≥3 → **direction = LONG**，强制开多
2. 多头触发成立 + 总分≥2 → **direction = LONG**，允许开多
3. 空头触发成立 + 总分≥3 → **direction = SHORT**，强制开空
4. 空头触发成立 + 总分≥2 → **direction = SHORT**，允许开空
5. 无触发条件 / 总分不足 → **direction = IDLE**，空仓观望
