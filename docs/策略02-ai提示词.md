# 二、趋势方向判断（必要条件，不计分）

# 三、评分系统（总分 = confidence，满分100）
必须逐项计算，不可跳过、不可漏项
## 固定参数阈值全程严格匹配：
EMA偏离阈值1.00%、EMA200价格偏离阈值1.60%、RSI区间35-65、K线影线阈值0.90%、前K量×1.8倍

---

【1. OI趋势匹配（25分）】
多头：
* OI非下降 且 OI变化率 > 0 → +25
空头：
* OI非下降 且 OI变化率 < 0 → +25
否则：
* +0（必须说明 OI趋势 + 变化率）

---

【2. EMA接近（10分）】
* 当前价格与EMA快线 ≤ 1.00%，价格贴近EMA → +10
否则 +0（说明价格超出EMA回踩阈值，偏离过大）

---

【3. EMA偏离度（22分）】
* 价格与EMA慢线偏离 ≤ 1.60%，未超出EMA200乖离阈值、不过度远离均线 → +22
否则 +0（说明价格偏离EMA200阈值过大，乖离超标）

---

【4. RSI区间（10分）】
多头：
* RSI 在 **35~65** 区间 且有向上倾向 → +10
空头：
* RSI 在 **35~65** 区间 且有向下倾向 → +10
否则：
* +0（必须写明当前RSI具体数值，说明是否超出35-65上下限）

---

【5. 成交量（23分）】
* 预计最终成交量 ≥ 上一根K线成交量 × 1.8倍 → +23
否则 +0（说明未达到放量倍数要求，未放量）

---

【6. K线确认（10分）】
同时满足2个条件才算有效加分：
1.  K线方向与趋势一致 / 存在明显动量K线
2.  多头K线下影线幅度≤0.90%、空头K线上影线幅度≤0.90%
满足全部 → +10
否则 +0
